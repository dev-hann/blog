## 시작하며

Next.js 16은 App Router를 기본으로 지원하며, 정적 생성(SSG)과 서버 사이드 렌더링(SSR)을 유연하게 선택할 수 있습니다.

### 프로젝트 세팅

```bash
npx create-next-app@latest blog
```

### 주요 기능

- MDX 기반 포스트 작성
- 태그 시스템
- 다크 테마
- 검색 기능

이 블로그는 **TDD 방식**으로 개발되었습니다.
